# High Sierra 10.13.1 and above.

Backup of EFI Folder.

What works:
* Almost everything.

What does **NOT** work:
* Native motherboard Audio, but could be patched. 
* IGFx, works with SMBIOS 18.3 and latest version of High Sierra.
